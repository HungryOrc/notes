# Compiler (含有 Preprocessor, Compiler, Optimizer, Linker)

Compiler 是把 source code 转化为 exexutable binary 的.

Compiler 里面含有：
* Preprocessor
* Compiler
* Optimizer
* Linker

## Preprocessor
Preprocessor 做：
* `#include` files
  * code里使用 `#include ...` directive 的时候，source file 会把 被include的file的全部内容 直接复制粘贴到 `#include ...` directive 所在的地方
